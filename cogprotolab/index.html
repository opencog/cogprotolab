command prompt use instructions
1.
Type a command into command prompt and press [enter] to send it to CogServer shell.
2.
The command is evaluated and its output is brought back to the command prompt interface.
3.
Visualizer aside is automatically being updated to reflect the output of currently active visualization script.
visualizer navigation instructions
1.
Point and drag inside the central oval to scroll the visible contents.
2.
Point and drag orbiting ovals around the central oval to rotate them.
3.
Point and drag orbiting ovals in or out of the central oval area to zoom them in or out.
